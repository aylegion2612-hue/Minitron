# Minitron Landing

Static Phase 8 landing page for download distribution.

## Files
- `index.html` - OS detection + download button routing.
- `landing-config.js` - repository config used for GitHub release asset URLs.

## Setup
1. Set `window.MINITRON_REPO` in `landing-config.js` (for example `org/minitron`).
2. Deploy `landing/` as static site on Vercel.
