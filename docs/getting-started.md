# Minitron Getting Started

## Install
1. Download installer from the landing page.
2. Run installer for your OS:
   - Windows: `.exe`
   - macOS: `.dmg`
   - Linux: `.AppImage`

## First Run
1. Open Minitron.
2. Complete the Onboarding wizard.
3. Verify your API key.
4. Start chatting in the Chat tab.

## Core Checks
- Rules tab: create at least one safety rule.
- Skills tab: install one skill.
- Memory tab: run a memory query.
- Updates tab: check pending updates.
